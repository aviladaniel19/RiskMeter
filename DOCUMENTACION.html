🧩 Estructura de módulos

Módulos base (Teoría del Riesgo — Módulos I y II)

  • Módulo 1: Análisis técnico e indicadores (SMA, EMA, RSI, MACD, Bollinger, Estocástico)
  • Módulo 2: Rendimientos y propiedades empíricas (Jarque-Bera, Shapiro-Wilk, Q-Q plot)
  • Módulo 3: Volatilidad condicional (EWMA, ARCH, GARCH, EGARCH)
  • Módulo 4: CAPM y riesgo sistemático (Beta OLS, Security Market Line)
  • Módulo 5: VaR y CVaR (Paramétrico, Histórico, Montecarlo + Backtesting Kupiec)
  • Módulo 6: Optimización de Markowitz (Frontera eficiente, QP, no negatividad)

Módulos nuevos (Extensiones al diseño original)

  • Módulo 7: Señales automáticas (Panel semáforo con 5 indicadores cruzados)
  • Módulo 8: Contexto macro y benchmark (Alpha Jensen, Tracking Error, Information Ratio)
  • Módulo 9: Renta fija (Nelson-Siegel, duración, convexidad)
  • Módulo 10: Opciones (Black-Scholes, Greeks, volatilidad implícita)
  • Módulo 11: Stress testing (Escenarios extremos, shocks de tasa y volatilidad)

🔗 Endpoints principales

15 Endpoints HTTP implementados

  • GET /activos — Lista de activos disponibles
  • GET /precios/{ticker} — Serie histórica de precios
  • GET /rendimientos/{ticker} — Estadísticas de rendimientos
  • GET /indicadores/{ticker} — Indicadores técnicos
  • GET /garch/{ticker} — Modelos de volatilidad
  • GET /capm — CAPM y métricas de riesgo sistemático
  • POST /var — Value at Risk y CVaR
  • POST /frontera-eficiente — Optimización de Markowitz
  • GET /alertas — Señales automáticas de trading
  • GET /macro — Indicadores macroeconómicos y benchmark
  • GET /renta-fija/curva — Curva de rendimientos Nelson-Siegel ★
  • POST /bono/duracion — Análisis de duración y convexidad ★
  • POST /opcion/precio — Valoración Black-Scholes con Greeks ★
  • POST /stress — Análisis de escenarios extremos ★

💻 Stack tecnológico

Backend Python + API REST

  • FastAPI — Framework web async de alto rendimiento
  • Pydantic v2 — Validación de datos tipada
  • SQLAlchemy — ORM para persistencia (SQLite)
  • yfinance + fredapi — Conexión a fuentes de datos financieros
  • arch, statsmodels, scipy — Cálculos econométricos
  • pandas, numpy — Manipulación de datos numéricos

Frontend + Infraestructura

  • Chart.js 4.4 — Gráficos interactivos en dashboard SPA
  • HTML5 / CSS3 — Interfaz sin dependencias de frameworks
  • Docker — Contenedorización con multi-stage build
  • GitHub Actions — CI/CD automatizado en cada push
  • Render / Railway — Despliegue en PaaS gratuito
